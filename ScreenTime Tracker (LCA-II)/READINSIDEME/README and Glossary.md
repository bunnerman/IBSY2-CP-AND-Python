# README and Glossary.md
This file acts as a glossary for popular app .exe names, use these process names in `groups.json` for categories to appear 

**NOTE:** *This only tracks the time spent actively looking on the screen, so some apps like Spotify (which are mainly spent in the background) are absent*

For apps not present, use **Task Manager** to find their process name while active

Groups need not match their name here, any custom name for the group can be used, process names in `groups.json` are case-insensitive

If you would rather use the following automatically, copy the content of `groups_default.json` and paste it in `groups.json`

If the name of a process is non descriptive, use `aliases.json` instead to set their names correctly
